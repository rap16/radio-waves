# 一.项目概述

“业余无线电考试助手” 是一款专为无线电爱好者（HAM）量身打造的 iOS 备考与实用工具应用。它不仅是一个刷题软件，更是一个聚合了摩尔斯电码练习、天线计算、频率查询等硬核功能的“口袋工具箱”。
本项目旨在解决业余无线电操作证书考试（A/B/C类）资料分散、练习体验差的痛点，帮助更多人轻松跨过门槛，走进这个充满温度的无线电通信世界。
开发方式：本项目基于 百度文心快码 (Baidu Comate) 辅助开发，采用 Swift + SwiftUI 原生架构，实现了从 0 到 1 的高效构建。

# 三.核心功能
1. 痛点解决：从“能用”到“好用”
市面上的题库软件往往界面陈旧、广告多、更新慢。
本应用提供：
- 全覆盖题库：内置最新 A/B/C 三类题库（1700+道题），支持离线使用。
- 科学刷题：提供顺序练习、随机练习、模拟考试三种模式，配合错题本和收藏夹，通过算法帮助用户查漏补缺。
- 深度解析：每道题均配备详细解析与配图，让用户“知其然更知其所以然”。

3. 场景延伸：不仅是备考，更是工具
区别于传统竞品“考完即删”的尴尬，本应用内置了 8大硬核工具，覆盖无线电操作的全生命周期：
摩尔斯电码训练器：不仅能查表，还能进行听抄练习。
天线计算器：帮助爱好者在户外架台时快速计算偶极子天线尺寸。
Q简语与呼号查询：通联时的必备速查手册。
通联日志 (QSO Log)：随时记录每一次珍贵的空中相遇。

# 四、 社会价值与公益初心 
“做一件对他人也有用的小事。”
1. 为什么做这款 App？
为什么要做这个 App 我是一名独立开发者，也是无线电爱好者。一路学习考试，我发现身边不少朋友卡在了“搜不到合适资料、题库更新不及时、练习体验不顺手”这几件小事上。
也是非常的机缘巧合，看到了百度文心快码 (Baidu Comate)举办的 CCF程序员大会码力全开：AI加速营。
于是趁着这次机会我想做一件对大家都方便的小事：把题库、练习、背题体验打磨好，让准备考试的人少走弯路，把时间真正用在理解知识和练手上。
这个项目没有大公司的资源，也没有花哨的运营，只有耐心打磨的功能与一点点爱。希望它既能帮到你，让大家把时间花在理解知识上，而不是与糟糕的软件搏斗，也能让更多人走进这门有温度的“业余无线电”。

2.为什么我说是有温度的“业余无线电”
在危机与灾害中的价值 在危机和自然灾害时期，当有线通信、手机和其他传统通信手段出现故障时，业余无线电常被用作紧急通信手段，协调救灾活动。与商业通信系统不同，业余无线电通常独立于可能在灾害期间失效的地面通信设施。它分散在一个没有短板的环境内（不会像手机基站那样容易超载），再加上台站数量大、爱好者技术全面且勇于奉献，使得业余无线电具备足够的健壮性来应对紧急情况下的通信需求。
业余无线电操作员在应急制作天线和电源方面经验丰富，当今多数设备甚至可以由汽车电池供电。操作员可以使用数百个频率，并能快速建立通信网络，把不同的机构连接在一起，提高互操作性。这为应急通信带来了很大便利。许多国家每年举办“户外通信日”，专门练习这类应急通信技能。
一些真实的案例
2004年9月2日，业余电台为天气预报员提供来自巴哈马的飓风弗朗西斯实时信息。
2004年12月26日，印度洋海啸摧毁安达曼群岛与外界的通信，业余电台迅速建立联系，协调救援。
2005年8月，飓风卡特里娜期间，超过1000名业余电台操作员在格尔夫海岸提供应急通信。
2008年“5·12”汶川地震，成都业余无线电应急通信网在地震发生不到3分钟后启动应急通信程序，通过短波/超短波台站与外界联络，支援救灾。
2017年，飓风“玛丽亚”袭击之后，红十字会请求向波多黎各派遣50名业余无线电操作员，以提供通信服务。
一起把小事做好 把题库做到可靠、把体验打磨得顺手，也许就是我能做的一点点社会价值。
推广业余无线电，就是在为社会储备一份“断网”时的通信火种。

# 五、 技术实现与 Comate 赋能
本项目采用 MVVM 架构，使用 SwiftUI 构建现代化 UI，底层使用 SQLite 直接操作保证高性能。在开发过程中，百度文心快码 (Comate) 发挥了关键作用：
1. Comate 提效亮点
1.SQL 逻辑优化：面对 1700+ 条数据的复杂查询（如“按章节筛选未做错题”），Comate 帮助编写了高效的 SQLite C-API 调用代码，避免了手动拼接 SQL 的繁琐与错误。
2.在产品设计的过程中，UI的实现比较头疼，但是通过 Comate 的代码补全，快速生成了大量 SwiftUI 的 List 和 Card 视图结构，相较于其它ai ide，个人觉得有将开发效率提升了 50% 以上。

2. 技术细节
数据层：封装 DatabaseManager 单例，直接调用 SQLite3 C API，实现毫秒级查询，内存占用极低。
UI层：自定义 BottomNavBar 和 HeaderView，实现极光渐变背景与玻璃拟态效果，视觉体验远超同类工具。
交互：全手势操作，支持平滑的转场动画与触感反馈。
